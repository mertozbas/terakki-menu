# Terakki Alaçatı — QR Menü

Tek sayfa React + Vite ile hazırlanmış QR menü uygulaması. Yemek, Bar ve Ot Festivali olmak üzere üç sekme içerir. TR/EN dil desteği vardır.

Canlı URL (GitHub Pages): `https://burjucaglar.github.io/terakki-menu/`

## Geliştirme

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

`main` dalına push edildiğinde `.github/workflows/static.yml` iş akışı otomatik olarak GitHub Pages'e yayınlar. Repo `Settings → Pages → Build and deployment → Source: GitHub Actions` ile yapılandırılmalıdır.
